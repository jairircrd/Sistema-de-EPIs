# Sistema de EPIs

## Project Overview

The Sistema de EPIs is a web application designed for the management of Personal Protective Equipment (PPE). This platform provides an efficient way for organizations to track, distribute, and manage their EPI inventory. It ensures that employees have access to the necessary protective gear and facilitates compliance with safety regulations.

## Features
- **Inventory Management**: Keep track of all available EPIs, including their quantities and locations.
- **User Management**: Assign roles and permissions to employees, allowing for secure and organized access to the system.
- **Request System**: Enable employees to request EPIs as needed, with automated approval workflows.
- **Reporting**: Generate reports on EPI usage, inventory levels, and compliance metrics.
- **Notifications**: Send alerts for low stock levels or upcoming expiration dates for certain equipment.

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Docker, Heroku

## Installation Instructions
1. Clone the repository: `git clone https://github.com/jairircrd/Sistema-de-EPIs.git`
2. Navigate to the project directory: `cd Sistema-de-EPIs`
3. Install the necessary packages: `npm install`
4. Start the server: `npm start`

## Usage
Access the application at `http://localhost:3000` in your web browser after starting the server.
